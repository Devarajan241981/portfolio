# Jai Chandra Kumar Devarajan — Portfolio

Live: [viewmyportfolio.xyz](https://viewmyportfolio.xyz)

Personal portfolio site built with Next.js 16 (App Router) and Tailwind CSS v4 — dark/light themes, an animated hero, a draggable project showcase, and an interactive falling-code easter egg.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.

## Stack

- Next.js 16, React 19, TypeScript
- Tailwind CSS v4
- Framer Motion
